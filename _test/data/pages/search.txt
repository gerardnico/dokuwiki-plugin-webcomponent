====== WebComponent - Search ======


===== About =====
''search'' is a component that implement the [[doku>search|dokuwiki search form]] in a [[https://getbootstrap.com/docs/4.0/components/navbar/#forms|Bootstrap format]].

It must be in a [[navbar|navbar component]].
